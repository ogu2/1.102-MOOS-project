The ConstantDepth Task

Description

Attempt to obtain specified depth and hold it. Note this task does NOT control thrust.

Publishes

  • EndFlags

Subscribes

  • StartFlags
  • NAV_DEPTH
  • NAV_PITCH

Configuration Block

  • Common Task Parameters

  • Depth

    • Syntax: Depth = Depth (m)
    • Example: Depth =5 go to 5m of depth

Example Block

Task = ConstantDepth
{
	Name = KeepDepth
	Priority = 3
	InitialState = ON
	StartFlag = MissionStart
	FinishFlag = EndMission
	TimeOut = NEVER
	Depth = 10.0
}
