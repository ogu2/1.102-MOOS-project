The Time Out Task

Description

This task allows time temporal control within the MOOS suite. At the specified time out after going active the task asserts it End and Event flags. Typical uses of this task may involve delaying the start of another task or specifying an overall mission timeout.

Publishes

  • EndFlags
  • EventFlags

Subscribes

  • StartFlags
