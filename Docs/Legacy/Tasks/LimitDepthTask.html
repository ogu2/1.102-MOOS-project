Limit Depth Task

Description

This task prevents the vehicle from exceeding a script specified depth. The Event flags are published when this depth is exceeded.

Publishes

  • EndFlags
  • EventFlags

Subscribes

  • StartFlags
  • AUV_DEPTH
  • DEPTH_LIMIT

Configuration Block

  • Common Task Parameters

  • MaximumDepth

    • Syntax: MaximumDepth = MaxDepth
    • Meaning: Set the maximum permissble depth
    • Example: MaximumDepth = 50 set max depth to 50m
