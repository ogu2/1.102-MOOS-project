The ConstantHeading Task

Description

Attempt to obtain specified heading and hold it. Note this task does NOT control thrust.

Publishes

  • EndFlags

Subscribes

  • StartFlags
  • NAV_YAW

Configuration Block

  • Common Task Parameters

  • Heading

    • Syntax: Heading = Heading (deg clockwise)
    • Example: Heading =90 head East

Example Block

Task = ConstantHeading
{
	Name = SOUTH
	Priority = 3
	InitialState = OFF
	TimeOut = 30
	StartFlag = BeginSouthTrack
	FinishFlag = BeginWestTrack

	Heading = 180.0


}
