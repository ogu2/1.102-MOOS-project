The Holding Pattern Task

Description

Hold a combination of Thrust Rudder and Elevator to constant values. If a degee of freedom (eg Elevator) is not set then it is not controlled.

Publishes

  • EndFlags

Subscribes

  • StartFlags

Configuration Block

  • Common Task Parameters

  • Thrust

    • Syntax: Thrust = Thrust [0-100]

  • Rudder

    • Syntax: Rudder = Angle (Deg)
    • Example: Rudder =5

  • Elevator

    • Syntax: Elevator = Angle (Deg)
    • Example: Elevator =0

Example Block

Task = HoldingPattern
{
	Name = FORWARD
	Priority = 3
	InitialState = OFF
	TimeOut = NEVER
	StartFlag = BeginNorthTrack
	Thrust = 10.0
}
