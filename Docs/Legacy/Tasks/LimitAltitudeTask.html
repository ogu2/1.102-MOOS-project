Limit Altitude Task

Description

This task prevents the vehicle from dropping below a script specified minimum altitude. The Event flags are published when this depth is exceeded.

Publishes

  • EndFlags
  • EventFlags

Subscribes

  • StartFlags
  • NAV_ALTITUDE

Configuration Block

  • Common Task Parameters

  • MinimumAltitude

    • Syntax: MinimumAltitude = MinAltitude
    • Meaning: Set the minimum permissble altitude
    • Example: MinimumAltitude = 5 set min altitude to 5m

Example Block

Task = LimitAltitude
{
	Name = LimitAltitude
	Priority = 1
	InitialState = ON
	TimeOut = NEVER

	MinimumAltitude = 5
}
